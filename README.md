# Portfolio Website - Arya Muhammad Duta Syafinda

Personal portfolio website showcasing projects, education, and experience.

## Features
- Responsive design
- Project showcase
- Admin editing capabilities
- Local storage for data persistence

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- AOS (Animate On Scroll)

## Setup
1. Clone this repository
2. Open `index.html` in your browser
3. For admin access, add `?admin=true` to the URL
